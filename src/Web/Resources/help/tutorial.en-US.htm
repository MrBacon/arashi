All the pages of the Control Panel contains the same elements to help you navigate to & from the different pages.

The first important element is the Site Switcher.

It shows the current site that you can manage, and if you move the mouse pointer over it, it shows a list of all the available sites to switch to, and a link to view the current frontend site and a link to create a new one.

Another element is the little User panel in the top right corner of the page.

This little panel shows the name of the current logged user as a link; click on it to redirect to the page where you can manage the user detail infos.

The Log off link will disconnect from this control panel and redirect to the login page.

In all the pages of the Control Panel, except for the home, after the Site Switcher will be another important element: the Breadcrumbs.

Breadcrumbs show you where you have been and wich pages you have navigated from the home to the current page; each page name in the breadcrumbs is a link that will let you return quickly to that page.


Moving the pointer over the Home link, it will show a panel with the same links as the buttons of the Control Panel home page; this panel offers you a easy way to move from a page to another without the necessity to go back to the home page.
