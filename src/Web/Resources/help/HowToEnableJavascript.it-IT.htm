Come attivare JavaScript nel browser?

Per visualizzare correttamente il Pannello di Controllo di Arashi è necessario che JavaScript sia attivo nel browser. A tal fine, procedi come descritto di seguito a seconda del browser che utilizzi:


Internet Explorer (6.0)

  1. Seleziona "Strumenti" nel menu superiore.
  2. Scegli "Opzioni Internet".
  3. Fai clic sulla scheda "Protezione".
  4. Fai clic su "Livello personalizzato".
  5. Scorri fino alla sezione "Esecuzione script".
  6. Alla voce "Esecuzione script attivo", seleziona "Attiva" e fai clic su OK.

Netscape Navigator (4.8)

  1. Seleziona "Modifica" nel menu superiore.
  2. Scegli "Preferenze".
  3. Scegli "Avanzate".
  4. Scegli "Script e Plug-in".
  5. Seleziona la casella di controllo "Abilita JavaScript" e fai clic su OK.

Mozilla Firefox (1.0)

  1. Seleziona "Strumenti" nel menu superiore.
  2. Scegli "Opzioni".
  3. Scegli "Proprietà Web" nella navigazione di sinistra.
  4. Seleziona la casella di controllo accanto ad "Abilita JavaScript" e fai clic su OK.

Mozilla Firefox (1.5)

  1. Seleziona "Strumenti" nel menu superiore.
  2. Scegli "Opzioni".
  3. Scegli "Contenuto" nella navigazione di sinistra.
  4. Seleziona la casella di controllo accanto ad "Abilita JavaScript" e fai clic su OK.

Apple Safari (1.0)

  1. Seleziona "Safari" nel menu superiore.
  2. Scegli "Preferenze".
  3. Scegli "Protezione".
  4. Seleziona la casella di controllo accanto a "Enable JavaScript".

Non dimenticare che l'aggiornamento del browser o l'installazione di un nuovo software di protezione o patch di protezione può incidere sulle impostazioni JavaScript. Inoltre, se JavaScript è stato impostato su "Richiesta", è possibile che venga visualizzato un avviso relativo allo scaricamento di script potenzialmente pericolosi.
